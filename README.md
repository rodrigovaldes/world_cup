# World Cup is here!

This application helps to show some facts about our national soccer teams. Please find it running here: https://rodrigovaldes.shinyapps.io/Fifa/

## How to replicate?

1. Run preprocessing.py (python 3.6)
2. Use Fifa.Rmd

## Where did you get the data?

All data was compiled by Ágoston Török, available here, https://www.kaggle.com/agostontorok/soccer-world-cup-2018-winner/data
